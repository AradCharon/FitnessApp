# My Fitness App

یک برنامه تمرینی ساده برای مدیریت تمرینات **قدرتی (Strength)** و **هوازی (Cardio)**.  
این پروژه به دو زبان **Python** و **Java** نوشته شده و کاملاً بر اساس اصول **OOP (Object-Oriented Programming)** طراحی شده است.

---

## ویژگی‌ها

- ثبت تمرینات Strength و Cardio
- نمایش تمرین‌ها و کالری مصرف شده
- مشاهده آمار کلی تمرین‌ها و زمان کل
- طراحی کلاس‌بندی شده با OOP برای مدیریت بهتر کد

---

## زبان‌ها

این پروژه هم در **Python** و هم در **Java** پیاده‌سازی شده است و از کلاس‌ها و مفاهیم OOP برای مدیریت داده‌ها و تمرین‌ها استفاده می‌کند.
